# Linux

## 1、Linux简述

1. 是一个操作系统，作者林纳斯，诞生于1991年。
2. 开源免费，支持多种硬件平台（游戏机、机顶盒等）。
3. 多用户和多任务。
4. 安全性高。

> 常见版本

1. `RedHat` 的免费版本`CentOS`
2. `Debian` 比较适合高级用户，不适合新手
3. `Ubuntu` `Debian`衍生版，含有图形化界面

## 2、Linux目录

1. `/bin` 脚本命令目录
2. `/boot` 启动目录
3. `/dev` 设备目录
4. `/etc` 配置文件目录
5. `/home` 用户目录
6. `/lib` 库文件目录
7. `/lib64` 64位机器的库文件目录
8. `/media` 鼠标、音响等设备目录，
9. `/mnt` 磁盘挂载目录
10. `/opt` 安装软件的目录
11. `/proc` 内存映射目录
12. `/root` 管理员目录
13. `/run` 运行服务的目录
14. `/sbin` 高级命令目录
15. `/srv` 运行服务产生的文件目录
16. `/sys` 系统目录
17. `/tmp` 临时文件目录
18. `/usr` 用户目录
19. `/var` 额外产生日历的目录
