# Python

## 1、`Python` 简介

`Python` 诞生于 1989 年，2008 年发布`Python3`

### `Python` 能做什么？

`Web` 开发、网络爬虫、数据分析、自动化测试、机器学习、人工智能......

### 需要学习哪些内容？

- `Python` 基础入门

基础语法、高级语法、面向对象

- `Python` 语法进阶

进程与线程、正则表达式

- `Python` 办公自动化

`Word`、`Excel`、`PPT`、邮件自动化，文件自动化管理

- 数据库

`MySQL`、`Redis`、`MongoDB`

- `爬虫`

爬虫基础、`Selenium`可视化爬虫框架、`Scrapy`异步爬虫框架、反爬与反反爬

- 后端框架

`Flask`、`Django`
